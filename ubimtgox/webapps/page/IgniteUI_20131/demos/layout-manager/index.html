Layout Manager

The Layout Manager component provides different responsive layout modes, which do not use tables. The supported algorithms include a border layout, which has different regions - left, right, center, header and footer; a grid layout, with support for col and row spans; a vertical as well as flow layouts; and a column layout which can be used with css classes only
